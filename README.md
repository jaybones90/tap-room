# Tap Room

This is a simple Angular application that keeps track of beers in a keg and alerts when kegs drop below a certain limit. Basic CRUD functionality with Angular.

### 
* Clone repo
* `gulp serve`

### No known bugs at this time

### Technologies Used
* Angular 2
* Bootstrap
* NPM
* Gulp
